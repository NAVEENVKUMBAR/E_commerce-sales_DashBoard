E-Commerce Sales Dashboard (Excel)

🎯 Project Overview

 	This project is an interactive Excel dashboard designed to analyze and visualize e-commerce sales data for the years 2024–2025. The dashboard provides actionable business insights by transforming raw transactional data into meaningful visual analytics.
 	The goal of this project is to demonstrate data analysis, business intelligence, and dashboard design skills using Microsoft Excel.

📂 Dataset Description
The dataset contains 5,000 sales records with the following key fields:
  Order ID
  Order Date
  Customer Name
  Region & City
  Category & Sub-Category
  Product Name
  Quantity
  Unit Price
  Discount
  Sales
  Cost
  Profit
  Profit Margin
  Payment Mode
  Year & Month

🛠️ Tools & Techniques Used

  Microsoft Excel
  Pivot Tables
  Pivot Charts
  Slicers
  Dashboard Design
  Business Insight Generation

📊 Key Features

✅ Interactive dashboard with slicers
✅ Region-wise sales analysis
✅ Category performance tracking
✅ Profit and margin analysis
✅ Payment mode insights
✅ Business insights summary
✅ Clean and structured data model

📈 Business Insights Generated

The dashboard helps answer important business questions such as:
Which category generates the highest profit?
Which region performs best?
Which are the top performing product?
Which payment modes customers prefer
Monthly and yearly performance trends
